# 🐧 Linux Komutları

## ls

Bulunulan dizindeki dosya ve klasörleri listeler.

### Örnek

ls
ls -la

## pwd

Bulunulan dizinin tam yolunu gösterir.

### Örnek

pwd

## cd

Dizin değiştirmek için kullanılır.

### Örnek

cd /home
cd ..

## cat

Dosya içeriğini görüntüler.

### Örnek

cat user.txt

## grep

Metin içinde arama yapar.

### Örnek

grep root passwd.txt
