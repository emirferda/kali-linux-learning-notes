# 🐺 Lupin One Lab Notes

## Overview

Lupin One is a cybersecurity lab environment used for learning penetration testing methodologies and security concepts in a controlled setting.

---

## Objectives

During this lab I practiced:

- Information Gathering
- Network Enumeration
- Service Identification
- Basic Vulnerability Research
- Linux System Interaction

---

## Tools Used

- Kali Linux
- Nmap
- Metasploit Framework
- Linux Terminal

---

## Learning Outcomes

This lab helped me better understand:

- The importance of reconnaissance
- How information gathering affects later stages
- Basic penetration testing workflow
- Linux command line usage
- Security assessment methodology

---

## Key Takeaways

- Enumeration is one of the most important phases.
- Small pieces of information can lead to larger findings.
- Understanding attack techniques helps improve defense strategies.
- Practical labs are valuable for developing cybersecurity skills.
