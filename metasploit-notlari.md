# 🚀 Metasploit Notes

## What is Metasploit?

Metasploit is a penetration testing framework used for security research and vulnerability validation in authorized environments.

---

## Common Commands

### Search

```bash
search smb
```

Searches for available modules.

---

### Use Module

```bash
use exploit/windows/smb/example
```

Selects a module.

---

### Show Options

```bash
show options
```

Displays required parameters.

---

### Run

```bash
run
```

Executes the selected module.

---

## Learning Notes

- Metasploit is widely used in security testing.
- Understanding vulnerabilities is important for defense.
- Always use tools in authorized lab environments.
